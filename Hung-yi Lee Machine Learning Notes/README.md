# Hung-yi Lee Machine Learning Notes

机器学习的本质是如何挑选一个合适的模型(Function)。

**机器学习的 Training 三步骤：**
* Step1 : 有一系列模型 Model(A set of function)
* Step2 : 通过训练数据，优化模型(Goodness of function f)
* Step3 : 挑选一个最佳模型(Pick the "Best" Function)

**机器学习的 Testing 步骤：**

* 使用 Training 的模型，做推理(预测)
